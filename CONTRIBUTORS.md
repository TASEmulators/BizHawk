## How to Contribute

If you wish to contribute to BizHawkRafaelia:

1. Fork the repository
2. Create a feature branch
3. Make your changes with proper attribution
4. Submit a pull request with detailed description
5. Ensure all changes comply with applicable licenses

## Attribution Requirements

When using or modifying this software:

1. Maintain this CONTRIBUTORS.md file
2. Add your name and contributions when making significant changes
3. Respect all original licenses (MIT, GPL, BSD, etc.)
4. Provide clear attribution for any third-party code
5. Do not claim authorship of code you did not write
